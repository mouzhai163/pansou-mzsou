# mzsou
一个基于pansou的转存系统,支持百度网盘,夸克网盘,UC网盘,迅雷网盘转存.<a href="https://mzsou.com">查看Demo</a>

  ## 核心功能

  - 全网盘搜索：聚合多个网盘/资源渠道的搜索结果，支持链接、提取码等信息的智能识别。
  - 一键转存：对搜索结果直接发起转存，当前支持常用网盘（如百度、夸克、UC、迅雷等）。转存成功后返回可分享链接和提取码。
  - 自动过期清理：已转存资源到期自动清理，避免堆积和资源过期占用。
  - 缓存与复用：命中过期前的转存结果时直接返回，提升响应速度并减少重复操作。

  ## 用户端体验

  - 简洁搜索页：输入关键词快速获取结果，点击即可发起转存或提取。
  - 提取弹窗：展示分享链接、提取码、有效时间，提供一键复制和跳转。
  - 登录支持：支持游客访问（可配置），也可启用登录后使用。

  ## 后台管理

  - 登录/注册/邮件验证：内置认证体系，支持邮箱登录与验证。
  - 用户管理：查看、搜索、封禁/解封、修改角色、重置密码等。
  - 转存设置：按网盘开关转存、配置自动清理时间、控制前端展示的网盘类型。
  - 站点设置：站点名称、描述、访客权限、注册开关、广告与自定义脚本等。
  - 账号管理：维护各网盘的授权/凭证，便于转存时选择可用账号。
  - 广告与公告：自定义公告/广告内容与到期时间，控制展示。
